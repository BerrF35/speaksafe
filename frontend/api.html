POST /analyze

Submit an audio file for AI detection analysis. Returns a job ID for async result polling.

POST https://api.speaksafe.io/v1/analyze
Request parameters
ParameterTypeDescription
filemultipart/form-dataAudio file. Supported: WAV, MP3, OGG, FLAC, M4A, WebM. Max 25 MB.
segment_analysisbooleanOptional. If true, returns per-segment scores for hybrid detection. Default: false.
include_spectrogrambooleanOptional. If true, returns a base64 PNG spectrogram in the result. Default: false.
Example – cURL
curl -X POST https://api.speaksafe.io/v1/analyze \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -F "[email protected]" \
  -F "segment_analysis=true"
import requests

with open("sample.mp3", "rb") as f:
    resp = requests.post(
        "https://api.speaksafe.io/v1/analyze",
        headers={"Authorization": "Bearer YOUR_API_KEY"},
        files={"file": f},
        data={"segment_analysis": "true"}
    )
job = resp.json()
print(job["job_id"]) # e.g. "a1b2c3d4-e5f6-..."
const form = new FormData();
form.append("file", audioFile); // File object
form.append("segment_analysis", "true");

const resp = await fetch("https://api.speaksafe.io/v1/analyze", {

const { job_id } = await resp.json();

GET /results/{job_id}

Poll for analysis results. Returns 202 while processing, 200 with full result payload once complete.

GET https://api.speaksafe.io/v1/results/{job_id}
